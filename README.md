## 📌 About this software

Modern AI agents often lose track of previous corrections or specific task requirements. This leads to repetitive errors and wasted time. Otto solves this problem by acting as a behavior layer that sits between your AI agents and their execution environment. It records your feedback and forces the agents to apply your lessons in future tasks.

This application provides a stable memory bank for your automated workflows. It ensures that once you fix an agent, the agent stays fixed.

## ⚙️ Core benefits

* **Consistent Behavior:** Your agents keep your standards across all sessions.
* **Persistent Correction:** One correction teaches the agent forever.
* **Human Control:** You remain the primary authority in the workflow.
* **Local Privacy:** Your data stays on your machine.

## 🧠 Using the memory layer

Otto operates automatically once it detects an active agent session. You do not need to configure complex settings to make it work.

1. Start your preferred coding agent or AI assistant.
2. When the agent makes a mistake, correct it using your standard prompts.
3. Otto captures the correction in the background.
4. Moving forward, the app injects these historical lessons back into the agent context whenever a similar task arises.

## 🛠️ Managing agent skills

You can view the list of corrections and skills stored by otto within the dashboard. The dashboard lists every agent connected to the system. You can toggle specific rules on or off if you want to experiment with different agent behaviors. 

## 🛡️ Privacy and data safety

Otto uses a local-first architecture. This means your interaction logs and correction history remain on your hard drive. No external server collects or stores your specific agent instructions. You retain full ownership of your data at all times.

## 📝 Frequently asked questions

**Does this work with every AI agent?**
Otto supports most major local and cloud-based agents available today. It functions best with agents that accept system-level instruction buffers.

**My antivirus blocked the installation. What should I do?**
Security software often flags desktop applications that are not widely distributed. You can safely add an exception in your security settings to allow the installer to run.

**Can I clear the memory?**
Yes. Use the settings menu inside the application to wipe specific agent histories or perform a full factory reset of your learned behaviors.

## 🔧 Troubleshooting

* **App fails to open:** Verify that you have the latest version of Windows installed. Restart your computer if the installer hangs.
* **Agent does not remember corrections:** Ensure the otto application is running in your system tray before you start your agent session.
* **High resource usage:** If the app feels slow, close other unnecessary browser tabs or programs that share the same system memory.
